因子投资-公式与研究检查清单

QUANT RESEARCH NOTE / REFERENCE

这篇文章有两个用途：学习时统一公式口径，研究交付前逐项验收。所有公式默认收益率用小数表示；i 是股票，t 是时间，N 是当期股票数，T 是期数，K 是解释变量或因子数。凡出现“年化”，都必须同时写明原始频率和一年期数。

一、价格、收益与复利

1. 含现金分配的简单收益率

Ri,t=(Pi,t+Di,t−Pi,t−1)/Pi,t−1R_{i,t}=(P_{i,t}+D_{i,t}-P_{i,t-1})/P_{i,t-1} Ri,t​=(Pi,t​+Di,t​−Pi,t−1​)/Pi,t−1​

P 是每股价格，单位为元/股；D 是本期每股现金分配，单位同为元/股；R 是每期无量纲比例。若数据供应商已经提供含分红复权价，不能再加一次 D。横截面板的维度为 N×T。

2. 对数收益率

ri,t=ln⁡(1+Ri,t)=ln⁡(Pi,t∗/Pi,t−1∗)r_{i,t}=\ln(1+R_{i,t})=\ln(P^*_{i,t}/P^*_{i,t-1}) ri,t​=ln(1+Ri,t​)=ln(Pi,t∗​/Pi,t−1∗​)

P* 表示口径一致的总回报价格。对数收益可跨时间相加；组合横截面收益仍应以简单收益按权重相加。R 接近负一时，对数收益对极端亏损非常敏感；R 等于负一时没有有限对数值。

3. 累计与年化收益

R0:Tcum=∏t=1T(1+Rt)−1R_{0:T}^{cum}=\prod_{t=1}^{T}(1+R_t)-1 R0:Tcum​=t=1∏T​(1+Rt​)−1

Rann=(1+R0:Tcum)A/T−1R^{ann}=(1+R_{0:T}^{cum})^{A/T}-1 Rann=(1+R0:Tcum​)A/T−1

A 是一年包含的观测期数，例如日频常近似 252，月频为 12。累计收益是整个区间比例；年化收益是每年比例。短样本的年化会放大偶然结果，不能当作稳定预期。

二、风险、相关性与组合

4. 样本均值、方差与波动率

Rˉ=(1/T)∑t=1TRt\bar R=(1/T)\sum_{t=1}^{T}R_t Rˉ=(1/T)t=1∑T​Rt​

s2=(1/(T−1))∑t=1T(Rt−Rˉ)2s^2=(1/(T-1))\sum_{t=1}^{T}(R_t-\bar R)^2 s2=(1/(T−1))t=1∑T​(Rt​−Rˉ)2

σann=sA\sigma_{ann}=s\sqrt A σann​=sA​

均值和标准差的单位是每期收益率，方差单位是收益率平方。波动率平方根年化隐含相同频率分布和有限二阶矩等近似条件；存在强烈自相关时应从聚合收益直接估计，不能机械乘根号 A。

5. 协方差与相关系数

cov(X,Y)=(1/(T−1))∑t=1T(Xt−Xˉ)(Yt−Yˉ)cov(X,Y)=(1/(T-1))\sum_{t=1}^{T}(X_t-\bar X)(Y_t-\bar Y) cov(X,Y)=(1/(T−1))t=1∑T​(Xt​−Xˉ)(Yt​−Yˉ)

ρXY=cov(X,Y)/(sXsY)\rho_{XY}=cov(X,Y)/(s_Xs_Y) ρXY​=cov(X,Y)/(sX​sY​)

协方差单位是收益率平方，相关系数无量纲且介于负一与一。相关矩阵是 N×N；它描述线性共同变化，不给出因果方向。

6. 组合收益与方差

Rp,t=wt′RtR_{p,t}=w_t^{\prime}R_t Rp,t​=wt′​Rt​

σp2=w′Σw\sigma_p^2=w^{\prime}\Sigma w σp2​=w′Σw

w 和 R 都是 N×1，Σ 是 N×N 协方差矩阵，结果分别是一期组合收益和收益平方。权重必须与下一期收益的资产顺序一致；当期收盘后形成的权重不能假设以同一收盘价无摩擦成交。

三、绩效指标

7. Sharpe 比率

SRann=A Rˉe/s(Re)SR_{ann}=\sqrt A\,\bar R_e/s(R_e) SRann​=A​Rˉe​/s(Re​)

R(e) 是每期组合收益减同期限无风险收益。Sharpe 无量纲，但必须报告样本区间、频率、无风险利率、是否扣费以及置信区间。非独立同分布收益下，根号年化只是近似。

8. 信息比率与跟踪误差

TEann=A s(Rp−Rb)TE_{ann}=\sqrt A\,s(R_p-R_b) TEann​=A​s(Rp​−Rb​)

IRann=A (Rp−Rb)‾/s(Rp−Rb)IR_{ann}=\sqrt A\,\overline{(R_p-R_b)}/s(R_p-R_b) IRann​=A​(Rp​−Rb​)​/s(Rp​−Rb​)

R(b) 是同频基准收益。TE 单位是每年百分比，IR 无量纲。基准必须与投资范围、币种和再投资口径一致，否则主动收益没有清晰含义。

9. 最大回撤

Wt=∏u=1t(1+Ru)W_t=\prod_{u=1}^{t}(1+R_u) Wt​=u=1∏t​(1+Ru​)

DDt=Wt/max⁡0≤u≤tWu−1DD_t=W_t/\max_{0\le u\le t}W_u-1 DDt​=Wt​/0≤u≤tmax​Wu​−1

MDD=min⁡tDDtMDD=\min_t DD_t MDD=tmin​DDt​

W 是从一开始的净值，DD 与 MDD 是非正比例。也有人把最大回撤报告为正数绝对值，报告中必须声明符号。最大回撤高度依赖样本起止点，不能只给一个数而不展示路径和修复时长。

四、回归与统计推断

10. OLS

y=Xβ+εy=X\beta+\varepsilon y=Xβ+ε

β^=(X′X)−1X′y\hat\beta=(X^{\prime}X)^{-1}X^{\prime}y β^​=(X′X)−1X′y

y 是 T×1，X 是 T×K，β 是 K×1，残差 ε 是 T×1。若 X 包含常数列，第一项通常是截距。逆矩阵公式要求 X 满列秩；实际代码更适合数值稳定的最小二乘或 QR 分解，而非显式求逆。

11. t 统计量

tj=(β^j−βj,0)/SE(β^j)t_j=(\hat\beta_j-\beta_{j,0})/SE(\hat\beta_j) tj​=(β^​j​−βj,0​)/SE(β^​j​)

t 无量纲。它只相对于所采用的误差模型成立：普通同方差标准误、White 稳健标准误、聚类标准误和 Newey–West 标准误可以给出完全不同的判断。

12. Fama–MacBeth 风险价格

每期横截面回归：

Ri,t+1=λ0,t+xi,t′λt+ui,t+1R_{i,t+1}=\lambda_{0,t}+x_{i,t}^{\prime}\lambda_t+u_{i,t+1} Ri,t+1​=λ0,t​+xi,t′​λt​+ui,t+1​

时间平均：

λˉ=(1/T)∑t=1Tλ^t\bar\lambda=(1/T)\sum_{t=1}^{T}\hat\lambda_t λˉ=(1/T)t=1∑T​λ^t​

x 是决策时点可得的 N×K 特征或暴露，λ(t) 是 K×1 的当期价格。回归用未来一期收益，所以 x 与收益的时间对齐是核心。标准误来自 λ(t) 的时间序列，若其自相关则需 HAC 修正。

13. Newey–West 协方差

令 u(t)=x(t)e(t) 为 K×1 得分向量，截断滞后为 L，Bartlett 权重为 q(l)=1-l/(L+1)：

S=Γ0+∑l=1Lq(l)(Γl+Γl′)S=\Gamma_0+\sum_{l=1}^{L}q(l)(\Gamma_l+\Gamma_l^{\prime}) S=Γ0​+l=1∑L​q(l)(Γl​+Γl′​)

Γl=(1/T)∑t=l+1Tutut−l′\Gamma_l=(1/T)\sum_{t=l+1}^{T}u_tu_{t-l}^{\prime} Γl​=(1/T)t=l+1∑T​ut​ut−l′​

S 是 K×K 的长期协方差核心，需结合设计矩阵得到系数协方差。L 不是“越大越稳健”：过小遗漏相关，过大增加估计噪声。必须报告选择规则和敏感性。

14. GRS 联合 Alpha 检验

设 N 个测试资产对 K 个因子做 T 期时序回归，α 是 N×1 截距向量，Σ(e) 是 N×N 残差协方差，μ(f) 与 Σ(f) 是 K×1 因子均值和 K×K 因子协方差：

GRS=((T−N−K)/N) α^′Σ^e−1α^1+μ^f′Σ^f−1μ^fGRS=((T-N-K)/N)\, \frac{\hat\alpha^{\prime}\hat\Sigma_e^{-1}\hat\alpha} {1+\hat\mu_f^{\prime}\hat\Sigma_f^{-1}\hat\mu_f} GRS=((T−N−K)/N)1+μ^​f′​Σ^f−1​μ^​f​α^′Σ^e−1​α^​

在经典假设下它服从 F(N,T-N-K)。检验的零假设是全部 Alpha 同时为零。若 T 不足、N 很大或协方差接近奇异，统计量会不稳定；拒绝模型也不告诉我们哪一个替代模型正确。

五、因子预测与交易实现

15. IC 与 Rank IC

ICt=corri(xi,t,Ri,t+1)IC_t=corr_i(x_{i,t},R_{i,t+1}) ICt​=corri​(xi,t​,Ri,t+1​)

RankICt=corri(rank(xi,t),rank(Ri,t+1))RankIC_t=corr_i(rank(x_{i,t}),rank(R_{i,t+1})) RankICt​=corri​(rank(xi,t​),rank(Ri,t+1​))

相关在每个 t 的横截面 i 上计算，得到长度 T 的时间序列。IC 无量纲。必须说明收益期限、股票池、缺失值处理、中性化和极值处理。IC 均值显著不保证分层组合扣费后赚钱。

16. 换手率

考虑收益漂移后的再平衡前权重：

wi,tpre=wi,t−1post(1+Ri,t)/∑jwj,t−1post(1+Rj,t)w_{i,t}^{pre}=w_{i,t-1}^{post}(1+R_{i,t})/ \sum_j w_{j,t-1}^{post}(1+R_{j,t}) wi,tpre​=wi,t−1post​(1+Ri,t​)/j∑​wj,t−1post​(1+Rj,t​)

单边换手定义：

TOt=(1/2)∑i∣wi,tpost−wi,tpre∣TO_t=(1/2)\sum_i|w_{i,t}^{post}-w_{i,t}^{pre}| TOt​=(1/2)i∑​∣wi,tpost​−wi,tpre​∣

TO 是每期比例。若不除以二，得到双边成交权重；两种口径都可用，但成本系数必须匹配。多空组合还需明确按总敞口、净资产还是资本占用归一。

17. 交易成本与净收益

一个透明的基线模型是：

Ct=∑i(ci,tlinear∣Δwi,t∣+ci,timpact∣Δwi,t∣3/2)C_t=\sum_i(c_{i,t}^{linear}|\Delta w_{i,t}| +c_{i,t}^{impact}|\Delta w_{i,t}|^{3/2}) Ct​=i∑​(ci,tlinear​∣Δwi,t​∣+ci,timpact​∣Δwi,t​∣3/2)

Rp,tnet=Rp,tgross−Ct−BtR_{p,t}^{net}=R_{p,t}^{gross}-C_t-B_t Rp,tnet​=Rp,tgross​−Ct​−Bt​

线性项可以包含佣金、印花税和半价差，冲击项随订单规模非线性上升，B 是借券与融资成本。C、B 和收益都用每期净资产比例。A 股卖出印花税是方向不对称成本，不能给买卖两边套同一费率。

18. 常见组合约束

1′w=1\mathbf{1}^{\prime}w=1 1′w=1

li≤wi≤uil_i\le w_i\le u_i li​≤wi​≤ui​

Aw=bAw=b Aw=b

∥w−wbench∥1≤TOmax\|w-w^{bench}\|_1\le TO_{max} ∥w−wbench∥1​≤TOmax​

四类约束分别表示资金守恒、个股上下限、行业或风格等线性暴露约束、相对基准的换手预算。实际可交易集合还必须在优化前剔除当日无法买卖的证券；优化器给出权重不等于交易能够成交。

六、数据与回测检查清单

研究问题

  • [ ] 因子有一句可证伪的经济假设，而不是先看结果再命名。
  • [ ] 目标变量、预测期限、再平衡频率、基准和评价指标在看测试集前固定。
  • [ ] 明确区分风险补偿、行为偏差、制度摩擦和纯统计预测四类解释。

股票池与点时数据

  • [ ] 股票池按每个历史时点重建，包含后来退市和长期停牌的证券。
  • [ ] 价格、分红、拆并股和配股只使用一套一致的复权逻辑。
  • [ ] 财报使用首次公告日或当时可得版本，不用报告期末直接对齐收益。
  • [ ] 指数成分、行业分类、ST 状态、上市日期和退市日期均为历史版本。
  • [ ] 主板、创业板、科创板和北交所的涨跌停、上市初期制度按日期处理。
  • [ ] 缺失值、极值、负权益和一字板不是用全样本信息决定删除。

信号与组合

  • [ ] 每个原始字段有定义、单位、可得时点、更新频率和供应商口径。
  • [ ] 排名、标准化、去极值和中性化只用当期横截面信息。
  • [ ] 分组断点、组数、等权或价值加权、持有期和跳过期均明确。
  • [ ] 信号形成、下单、成交和收益计量的时间线可以画成一条无歧义的轴。
  • [ ] 停牌、涨跌停、最小交易单位和融券不可得时有明确的持仓迁移规则。

推断、稳健性与数据挖掘

  • [ ] 报告效应量、标准误、置信区间和样本长度，不只报告星号。
  • [ ] 重叠持有期和高频再平衡导致的自相关使用合适的 HAC 或聚类标准误。
  • [ ] 因子模型使用多组测试资产时，同时报告 Alpha 分布和联合检验。
  • [ ] 变量、样本和参数尝试次数有记录，并采用多重检验或独立验证控制。
  • [ ] 样本拆为训练、验证和最终测试；最终测试不参与特征和参数选择。
  • [ ] 替代定义、子时期、行业、规模、微盘股排除与发表后结果均被检查。

成本、约束与容量

  • [ ] 毛收益与净收益并列，成本参数注明日期、市场和单边/双边口径。
  • [ ] 佣金、税费、价差、冲击、借券、融资和未成交分别建模。
  • [ ] 个股、行业、风格、Beta、总敞口、净敞口和换手约束清楚。
  • [ ] 容量按成交额参与率、持仓天数和冲击曲线做情景分析。
  • [ ] 参数在保守、中性和乐观三种情景下均展示，结论不依赖最乐观情景。

可复现交付

  • [ ] 原始数据不被覆盖，清洗数据和特征数据有版本或内容哈希。
  • [ ] 代码固定随机种子、依赖版本、时区、交易日历和排序稳定规则。
  • [ ] 报告保存参数、样本筛选计数、中间表、图、日志和运行日期。
  • [ ] 每张表和图能追溯到脚本与输入，每个结论能追溯到表或检验。

七、发布前六个验收问题

  1. 问题是否先于结果？ 能否在隐藏最终测试集时，写出因子的经济机制、方向、期限和反证条件？
  2. 数据是否当时可得？ 任取一个历史交易日，能否列出每只股票为什么进入股票池、信号来自哪个已知版本？
  3. 实现是否可成交？ 信号形成到成交的时间线是否尊重公告、停牌、涨跌停、交易单位、做空和资金约束？
  4. 证据是否经得住统计审查？ 效应量、稳健标准误、联合检验、多重检验、样本外和发表后证据是否齐全？
  5. 收益是否经得住经济审查？ 在保守成本、容量和约束下，净收益是否仍大于机会成本并具有合理机制？
  6. 第三方能否复现？ 新环境中的研究者能否凭数据字典、版本、参数和脚本得到相同的关键数字？

只要其中一题回答“不能”，结论就应降级为探索性证据。六题全部通过也不代表未来收益有保证，只说明研究过程达到了可审计、可反驳和可复现的最低标准。
