因子投资-07-APT多因子模型与异象

QUANT RESEARCH NOTE / LESSON 07

市场风险不可能解释所有股票共同变化。利率、盈利、投资、价值和流动性等力量也可能系统性影响资产。APT 提供多因子定价的理论骨架，但它不自动告诉我们因子是谁。实证研究必须同时证明一个候选因子“造成共同运动”并且“在横截面被定价”。

本节解决什么问题

市场风险不可能解释所有股票共同变化。利率、盈利、投资、价值和流动性等力量也可能系统性影响资产。APT 提供多因子定价的理论骨架，但它不自动告诉我们因子是谁。实证研究必须同时证明一个候选因子“造成共同运动”并且“在横截面被定价”。

前置知识与学习目标

需要第 06 讲的 Beta、Alpha 和风险溢价。学完后你应该能：

  • 写出多因子收益生成式和期望收益式；
  • 区分共同因子、特质扰动和定价误差；
  • 解释风险补偿与错误定价的竞争假说；
  • 判断经验规律何时只能称为异象；
  • 用“共同运动 + 被定价”两道门槛审查候选因子。

直觉与手算

假设模型只有市场 M 和流动性 L 两个因子，预期因子溢价分别为每月 0.5% 与 0.3%。股票 A 的暴露为 β(M)=1.2、β(L)=0.5，股票 B 为 0.8、-0.2。忽略截距时：

\[ E[R_A^e]=1.2\times0.5\%+0.5\times0.3\%=0.75\% \]

\[ E[R_B^e]=0.8\times0.5\%-0.2\times0.3\%=0.34\% \]

A 的更高预期收益来自更高的已定价暴露。若 A 实际样本平均超额收益为 1%，模型定价误差为 0.25%。这个差异可能是真 Alpha、有限样本噪声、遗漏因子、错误暴露或数据问题，不能仅凭正数决定解释。

定义与公式

多因子收益生成式

对资产 i：

\[ R_{i,t}^e=\alpha_i+\beta_i^{\prime}f_t+\varepsilon_{i,t} \]

f(t) 是 K×1 共同因子收益，β(i) 是 K×1 暴露。共同因子使很多资产一起变化；ε 是给定因子后的特质部分。若模型正确定价测试资产，Alpha 向量应接近零。

多因子期望收益

APT 的无套利直觉是：在资产很多、特质风险可分散时，两个具有相同因子暴露的充分分散组合不应有不同价格；否则可构造近似无风险套利。因此：

\[ E[R_i^e]=\beta_i^{\prime}\lambda \]

λ 是 K×1 因子风险价格或溢价。APT 依赖无套利和因子结构，不像 CAPM 那样要求市场组合是唯一均值—方差有效组合；代价是理论本身没有唯一指定因子身份。

定价误差

\[ \alpha_i=E[R_i^e]-\beta_i^{\prime}\lambda \]

Alpha 是相对模型的误差。模型增加高度相关因子可能机械降低样本 Alpha，却未必增加经济解释；评价需要样本外、联合检验和简约性。

因子的两道门槛

  1. 共同运动： 因子收益应解释一组资产的协方差，暴露在经济上和统计上可辨识。
  2. 被定价： 暴露更高的资产应有系统性不同的平均收益，λ 在稳健推断和样本外仍有意义。

一个宏观变量可能影响所有股票但没有风险价格；一个股票特征可能预测个股收益却不形成共同协方差。两者都不能在不说明定义的情况下直接叫“风险因子”。

异象、风险与错误定价

异象是相对既定模型难以解释的稳定经验模式。风险解释认为高平均收益是坏状态下损失的补偿；行为解释认为投资者认知偏差导致价格偏离基本面；制度与摩擦解释强调卖空、融资、委托代理和交易限制。

竞争解释应给出不同可检验预测。例如风险补偿应与宏观坏状态、边际效用或系统损失相关；错误定价常预测在套利限制强、关注低或情绪极端处更强，并可能在公开和套利后衰减。

Python 最小实验

矩阵维度为 2 只资产×2 个因子，乘以 2×1 溢价，得到 2×1 预期超额收益。这里的 λ 是人为给定真值；实证中需要截面回归估计，并处理暴露测量误差。

A 股实现注意事项

  1. 共同因子要可投资或可复制。 宏观序列公布滞后、修订且不可交易，和股票模拟组合因子有不同解释。
  2. 卖空限制削弱套利。 多空异象的空头腿可能融券不可得或成本极高，样本 Alpha 未必能转化为套利。
  3. 壳价值与制度。 极小市值股票可能受上市资格、重组和退市制度影响，不应直接套用美国规模因子解释。
  4. 状态与结构变化。 注册制、外资流入、融资融券扩容会改变因子暴露、风险价格和套利能力。
  5. 因子构造重叠。 市值、流动性、波动和价格在 A 股高度相关；应检查相关、暴露、正交化顺序与经济定义。
  6. 微盘权重。 等权多空收益可能主要来自难交易微盘股。使用价值权重、排除极小股票和加入容量是必要稳健性。

常见错误与边界

  • 候选变量与收益相关就直接叫风险因子，没有检验共同协方差。
  • 因子能解释协方差就声称它有正风险溢价，没有横截面证据。
  • 用样本内最高 Sharpe 选择因子，再用同一样本报告显著性。
  • 把模型 Alpha 降低等同于模型更真实，忽略因子数量和过拟合。
  • 看到异象就二选一地归因风险或行为，未给竞争预测。
  • 认为无套利意味着现实中不存在价格偏离；有限资本和交易约束允许偏离长期存在。

APT 是框架而非因子清单。任何一组统计主成分都能描述共同运动，但旋转后的因子不一定有稳定经济身份；任何一组股票特征也可能拟合平均收益，但不一定代表系统风险。

原书与论文

原书第 2 章（PDF 文件页 27–96）讨论多因子模型和实证方法，第 7 章（291–377）进一步触及行为解释和前沿。

Ross（1976）是 APT 原始文献：DOI 10.1016/0022-0531(76)90046-6。Fama 与 French（1993）展示如何用可交易模拟组合描述股票与债券的共同收益变化：DOI 10.1016/0304-405X(93)90023-5。

练习

  1. 三因子溢价为 0.4%、0.2%、-0.1%，某资产暴露为 1.1、0.5、-0.3，计算预期超额收益。
  2. 给出一个“解释共同运动但未必被定价”的变量，以及一个“预测收益但未必是风险因子”的特征。
  3. 两个充分分散组合暴露完全相同，预期收益却相差 2%。写出近似套利的多空方向和现实约束。
  4. 设计一组能区分“低流动性收益是风险补偿”与“投资者忽略”的状态检验。
  5. 比较统计主成分因子与价值多空组合：可解释性、可交易性和稳定性各有什么差异？

校验：第 1 题为 1.1×0.4%+0.5×0.2%+(-0.3)×(-0.1%)=0.57%。

小结与下一步

APT 用无套利把预期收益写成多因子暴露与风险价格的内积。有效因子至少要解释共同运动并在横截面被定价。定价误差是相对模型的对象，异象则需要风险、行为、制度和数据问题之间的竞争检验。
