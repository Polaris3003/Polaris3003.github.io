因子投资-15-主流多因子模型

QUANT RESEARCH NOTE / LESSON 15

多因子模型试图用少量共同收益解释大量资产平均收益与协方差。模型越多、因子越多，不代表越接近真相：增加变量总能改善样本内拟合，却可能带来冗余、估计噪声和更差样本外表现。本讲用统一表比较八类主流模型。

本节解决什么问题

多因子模型试图用少量共同收益解释大量资产平均收益与协方差。模型越多、因子越多，不代表越接近真相：增加变量总能改善样本内拟合，却可能带来冗余、估计噪声和更差样本外表现。本讲用统一表比较八类主流模型。

前置知识与学习目标

需要第 06、07、10 讲与 11–14 因子定义。学完后你应该能：

  • 识别 CAPM、FF3、Carhart4、FF5、q-factor 等模型因子；
  • 理解盈利、投资和行为因子怎样扩展价值/动量；
  • 用 GRS、平均绝对 Alpha、因子冗余和简约性联合评价；
  • 区分风险模型、定价模型和业绩归因模型的用途；
  • 避免把美国构造机械复制为 A 股真模型。

直觉与手算

假设 25 个测试组合在 CAPM 下平均绝对月 Alpha 为 0.30%，加入规模和价值后降到 0.15%，再加动量降到 0.08%。拟合确实改善，但还需问：

  • 0.07 个百分点是否大于估计误差？
  • 新因子本身能否被旧因子解释，是否只是重新包装？
  • GRS 是否仍拒绝全部 Alpha 为零？
  • 测试组合是否包含模型最难解释的资产？
  • 样本外和扣成本后是否仍改善？

如果用 20 个因子把 25 个组合拟合得近乎完美，模型几乎没有留下反证空间。简约不是审美，而是保护样本外有效性的约束。

各模型增量在回答什么

FF3 把 CAPM 未解释的规模与价值共同收益显式化。Carhart4 加动量，特别适合基金归因，但不声称动量一定是风险。

Novy‑Marx 指出毛盈利能力与价值具有增量信息；不同文献的“NM4”组合可能包含市场、规模、价值、盈利或动量，引用时必须列因子而非只写缩写。

FF5 用稳健减弱盈利 RMW、保守减激进投资 CMA 扩展 FF3。HML 在样本中可能被其他因子吸收，但“回归冗余”不等于价值特征没有预测。

q-factor 用市场、规模、投资 IA、盈利 ROE，从企业投资的一阶条件解释预期收益。它和 FF5 因子高度相关但理论与构造不同。

Stambaugh–Yuan M4 把 11 个异象按共同收益聚成两组：MGMT 汇总净发行、应计、净经营资产、资产增长、投资等管理相关变量；PERF 汇总困境、O-score、动量、毛盈利和 ROA。它们是综合错误定价代理。

DHS/BF3 在市场因子外加入 FIN 与 PEAD：FIN 用管理层发行/回购响应持续错误定价，PEAD 用盈利公告惊喜捕捉注意不足下的短期漂移。

模型评价的四个维度

  1. 联合 Alpha： GRS 检验全部测试资产 Alpha 是否为零。
  2. 效应大小： 平均绝对 Alpha、Alpha 均方根和最大 Alpha。
  3. 因子冗余： 新因子对旧因子回归是否有稳定 Alpha；暴露是否只重命名旧风险。
  4. 简约与样本外： 因子数、参数数、数据需求、测试集表现和跨市场稳定性。

GRS 对测试资产和协方差敏感；平均绝对 Alpha 不提供显著性；简约模型可能遗漏，复杂模型可能过拟合。四者需要并列。

Python 最小实验

用固定随机种子生成五个测试资产，真实过程包含市场、规模、价值和动量。比较三个嵌套模型的平均绝对 Alpha。

真实过程已知，所以完整模型更好并不惊讶。现实模型比较没有“真因子标签”，必须使用预留测试资产、联合推断和经济机制。

A 股实现注意事项

  1. 美国因子文件的断点、交易所与会计字段不能直接套 A 股。
  2. MKT、SMB、HML 的不同本地构造会让同名模型产生不同 Alpha。
  3. q-factor 与 FF5 的盈利、投资变量对金融行业和累计财报口径敏感。
  4. MGMT、PERF、FIN、PEAD 需要历史发行回购、公告窗口和异象字段，点时数据成本高。
  5. A 股做空限制使行为因子的空头腿特别难复制；测试模型和交易模型要分开。
  6. 测试资产应包括行业、规模—价值、动量和单因子组合，不能只选模型构造组合。
  7. 模型比较跨越制度变化时，应做注册制、交易费用和板块分段。

常见错误与边界

  • 因子多、R² 高就判模型更好。
  • 用模型构造组合同时做唯一测试资产，产生近似自解释。
  • 只看个别 Alpha，不做联合检验。
  • GRS 不拒绝就宣布模型为真，忽略低功效。
  • 同名模型不列精确因子与构造版本。
  • 在同一样本尝试多个模型后只报告最佳者。
  • 把风险模型因子用于协方差控制的成功，误写成预期收益定价成功。
  • 美国样本结论无条件外推 A 股。

练习

  1. 为八个模型各写一个最可能失败的测试资产组合。
  2. 新因子使平均绝对 Alpha 从 0.15% 降到 0.12%，但自身 Alpha 不显著；给出保留与拒绝的证据要求。
  3. 比较 FF5 的 RMW/CMA 与 q-factor 的 ROE/IA：理论、公式和排序有何不同？
  4. 用同一组测试资产计算 CAPM、FF3、Carhart4 的 Alpha 分布与 GRS。
  5. 将模型选择过程放入滚动样本，禁止测试期参与选因子。

小结与下一步

多因子模型是可反驳的压缩工具，不是越长越好的因子清单。CAPM 到 FF、q-factor 和行为模型分别强调市场风险、经验共同收益、投资理论和系统错误定价。评价必须同时看联合 Alpha、效应大小、冗余、简约和样本外。
